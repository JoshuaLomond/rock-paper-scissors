# Rock Paper Scissors Game

- This is a simple Rock Paper Scissors game.

- Implemented using JavaScript, HTML and CSS.

## Table of Contents

- [Features](#features)
- [How to Play](#how-to-play)
- [License](#license)

## Features

- Player can choose between Rock, Paper, and Scissors.

- CPU generates a random choice.

- The winner is determined based on the game rules.

- Scores are displayed for the player and CPU.

## How to Play

1. Open the game in a web browser.

2. Click on the Rock, Paper, or Scissors image to make your choice.

3. The CPU will randomly choose its option.

4. The winner and scores will be displayed.
